# Algoritmos y Estructuras de Datos - UTN

Repositorio de ejercicios, prácticas y trabajos realizados durante la materia **Algoritmos y Estructuras de Datos** de la carrera **Ingeniería en Sistemas de Información - UTN**.

## Lenguajes utilizados

- C
- Python

## Contenidos trabajados

- Fundamentos de programación
- Variables y tipos de datos
- Condicionales
- Bucles
- Funciones
- Punteros
- Vectores
- Matrices
- Estructuras (`struct`)
- Listas enlazadas
- Pilas y colas
- Árboles
- Grafos

## Organización del repositorio

- `Práctica primer parcial` - Ejercicios de preparación para el primer parcial
- `Práctica segundo parcial` - Ejercicios de preparación para el segundo parcial
- `TP4` - Funciones y punteros
- `TP5` - Vectores
- `TP6` - Matrices

## Objetivo

Este repositorio tiene como objetivo documentar mi progreso en el aprendizaje de programación, algoritmos y estructuras de datos utilizando principalmente C, incorporando Python para la implementación y práctica de estructuras de datos.
